## Installation
Run `npm install`

Run `json-server server/db.json` to start the backend server which contains the favorites list.  Navigate to `http://localhost:3000/` to view the database.

Run `ng serve` for a dev server. Navigate to `http://localhost:4200/`. The app will automatically reload if you change any of the source files.

# Open Weather Application

- Dynamic Weather Data Fetched from Open Weather Map API
- Weather Displayed in the form of Widgets
- Favorites List

# Angular Firebase Authentication

- Sign-in with Google
- Sign-in with username/password
- Register with username/password
- Email verification
- Forgot password
- Guard Routes with canActivate
- Prevent user to access sign in and sign up URL when a user is already logged in
- Handle logged-in user state with Local Storage
